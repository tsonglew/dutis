# Dutis(Rust Version)

A command-line tool to select default applications. It is a wrapper around [duti](https://github.com/moretension/duti).

## Installation

### Using HomeBrew

```shell
brew tap tsonglew/dutis https://github.com/tsonglew/dutis
brew install dutis
```

### Using Cargo

```shell
```

## Usage

```shell
dutis
```
